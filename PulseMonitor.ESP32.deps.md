> **Notice:** The project has been migrated to **PlatformIO**. 
> All ESP32-S3 dependencies are now automatically managed by the `platformio.ini` file in the `firmware/` directory.

### Legacy Arduino IDE / CLI Setup (Deprecated)

| Library | Source | Install Command |
|---|---|---|
| SparkFun MAX3010x Pulse and Proximity Sensor Library | Arduino Library Manager | `arduino-cli lib install "SparkFun MAX3010x Pulse and Proximity Sensor Library"` |
| ArduinoJson | Arduino Library Manager | `arduino-cli lib install "ArduinoJson"` |
| WebSockets by Markus Sattler | Arduino Library Manager | `arduino-cli lib install "WebSockets"` |
| ESP32 Arduino Core (for ESP32-S3, WiFi/WebSocket stack) | Arduino Board Manager | `arduino-cli core install esp32:esp32` |
